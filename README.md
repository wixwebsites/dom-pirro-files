# NATE MATTHIS FILES

Countdown site that reveals redacted photos at **March 25, 2026 10:25 AM**.

## How to use

1. Add your photos to the `photos/` folder
2. Run `python3 redact_faces.py ./photos ./redacted` to redact faces
3. Rebuild the site with embedded images
4. Open `index.html`

## Features
- Big countdown timer (HH:MM:SS)
- Release animation with flash, glitch, particles, and "RELEASED" stamp
- Scrollable photo gallery with filenames
- Click-to-popup lightbox
- Face redaction via OpenCV Haar cascades
- "End Countdown" button for testing (top-right)
